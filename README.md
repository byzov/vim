#Setup
1. `git clone git@github.com:byzov/vim.git  ~/.vim` OR get read-only git-url
1. `git clone http://github.com/gmarik/vundle.git ~/.vim/bundle/vundle`
1.  Add in .vimrc: `source ~/.vim/my.vim`
1. Launch Vim
1. `:BundleInstall`

#OR
Download last package
